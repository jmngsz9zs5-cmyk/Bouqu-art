# Bouqu-art

Site vitrine et catalogue pour une boutique de bouquets en satin.

## 🧩 Structure du projet

- `index.html` : page principale avec catalogue et panier.
- `styles.css` : styles de l'interface.
- `script.js` : logique de panier (stockée en `localStorage`) et interactions.

## ▶️ Lancer localement

Ouvrez `index.html` dans un navigateur moderne (Chrome, Firefox, Edge).

> Note : il s'agit d'une démonstration statique, il n'y a pas de paiement réel.
